This is a simple Holy Quran Reader Plugin.

== Description ==

"Read Holy Quran" Plugin has been made by using embeds code of quranflash.com that provides simple interface to user for reading holy quran easily from the website. such as you can read here full holy quran following each chapter. Enjoy

== Installation ==

1. Upload `Read Holy Quran` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place this shortcode`[holy-quran ]` in your page and post.

== Frequently asked questions ==

= Can i use this Plugin in any version of wordpress ? =

Yes. You can use this in any version of wordpress.

= How i can use this plugin ? =

Just active it and use shortcode "[holy-quran]".
